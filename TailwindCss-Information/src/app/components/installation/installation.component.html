Instalacja Tailwind CSS (Angular)

Dane zaczerpniete ze strony
  • Instalacja potrzebnych paczek:
    • npm install postcss --save-dev
    • npm install tailwindcss
  • Tworzenie pliku tailwind.config.js
    • npx tailwind init
  • module.exports = { content: ['./src/**/*.{html,ts}', './projects/**/*.{html,ts}'], theme: { extend: {}, }, plugins: [], };
  • Dodanie dyrektyw do pliku css (src/style.css)
    • @tailwind base;
    • @tailwind components;
    • @tailwind utilities;
